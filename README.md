## 功能介绍

### HyperlinkLabel
这是一个继承自 `Label` 的自定义控件，用于实现超链接功能。支持点击跳转到指定URL或执行指定函数，并具有鼠标悬停效果。

### DragDropFrame
这是一个支持拖拽文件的 `Frame` 控件。用户可以将文件拖拽到控件上，并触发回调函数。

### DragDropLabelFrame
这是一个支持拖拽文件的 `LabelFrame` 控件，与DragDropFrame类似。用户可以将文件拖拽到控件上，并触发回调函数。
